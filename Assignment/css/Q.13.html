Q.13 What are the ways to integrate CSS as a web page?

  • There are 3 ways to integrate CSS (Cascading Style Sheets) into a web page.

  • Inline CSS: Inline CSS is applied directly to individual HTML elements using the "style" attribute.

    • For Example
    Code : <p style="color: blue;">This is a paragraph with inline CSS.</p>
    Output :

    This is a paragraph with inline CSS.


  • Internal CSS: Internal CSS is placed within the <style> tags in the head section of an HTML document.

  • External CSS: External CSS is defined in a separate CSS file and linked to the HTML document using the <link> tag. This method allows you to keep your CSS code separate from your HTML, making it easier to maintain.
  • Using an external CSS file is considered a best for larger projects or websites
