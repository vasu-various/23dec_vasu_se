Q.11 How to center block elements using CSS1?

  • In CSS1, there is hvae method to center block elements horizontally. Here are a couple of techniques:
  • Auto Margins: You can utilize auto margins to horizontally center a block-level element.
  • This technique works because setting both left and right margins to auto will distribute the available horizontal space equally on both sides of the element, resulting in center alignment.


    • For Example :
    .element {
         margin-left: auto;
        margin-right: auto;
    }
