Q.9 Which property controls the image scroll in the background?

  • The CSS property that controls the image scroll in the background is called "background-attachment"

  • The background-attachment property can take the following values:

    • scroll:

    This is the default value, where the background image scrolls with the content as the user scrolls the page.

    fixed:

    The background image remains fixed in place while the content is scrolled. This means the image stays in the same position.

    inherit:

    The background-attachment value is inherited from the parent element.

    here a property is :
    background-attachment: fixed;
