Q.15 What are the external style sheets?

  • External CSS: External CSS is defined in a separate CSS file and linked to the HTML document using the <link> tag. This method allows you to keep your CSS code separate from your HTML, making it easier to maintain.
  • External style sheets provide a way to separate the presentation (styling) from the structure (content) of a web page.
  • By keeping the styles in a separate file, you can apply the same styles to multiple web pages, making it easier to maintain and update the overall look and feel of a website.
  • The external style sheet file has a .css extension and contains CSS code that specifies how HTML elements should be displayed.
  • To use an external style sheet, you need to include a <link> tag in the HTML <head> section that specifies the path to the CSS file.
