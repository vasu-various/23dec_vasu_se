Q.16 What are the advantages and disadvantages of using external style sheets?

    Advantages of using external style sheets:

  • With external style sheets, you can define and control the styles for multiple web pages from a single file.
  • If you need to make any style changes, you can update the external style sheet, and it will automatically apply those changes to all pages linked to it.
  • This separation improves the maintainability of your code.

    • Disadvantages of using external style sheets:

  • If the external style sheet fails to load due to network issues or incorrect file path, it can lead to the loss of styling for your web pages.
  • External style sheets apply styles globally to all elements on a web page or across multiple pages. While this can be an advantage for maintaining consistency, it can also be a disadvantage if you need to apply specific styles to individual elements or pages. In such cases, inline or embedded styles may be more appropriate.
  • If you have a small, simple website with just a few pages and minimal styling requirements, using an external style sheet may introduce unnecessary complexity.
