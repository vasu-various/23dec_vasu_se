Q.2 What are the disadvantages of CSS?

  • CSSis a powerful language for styling web pages, but it also has its limitations and disadvantages.
  • CSS can become complex and difficult to manage, specially in large projects.
  • Different web browsers may interpret CSS rules slightly differently.
  • This can create compatibility issues and require additional effort to ensure consistent styling across different platforms and devices.
  • CSS can affect the performance of web pages, especially when stylesheets become large and complex.
