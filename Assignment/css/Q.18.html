Q.18 What are the media types allowed by CSS?

  • It provides several media types that allow you to define different styles for different types of devices and media.
  • all: This is the default media type, which applies to all devices and media.
  • screen: This media type is used for computer screens, tablets, smartphones, and other similar devices.
  • tv: This media type is used for television-type devices.
  • speech: This media type is used for speech synthesizers and screen readers that read the document.
