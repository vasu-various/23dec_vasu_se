Q.8 What is the use of the background-position property?

  • It is primarily used in CSS (Cascading Style Sheets) to control the placement of a background image.
  • The background-position property accepts values that define the horizontal and vertical position of the background image.
  • These values can be specified in different units such as pixels (px), percentages (%), or keywords.

    • For Example :


    here we use a
    background-position: right top;
