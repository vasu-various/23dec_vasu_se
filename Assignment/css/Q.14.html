Q.14 What is embedded style sheets?

  • Embedded style sheets is a embedded CSS
  • Embedded CSS is also known as a internal CSS
  • Internal CSS is placed within the <style> tags in the head section of an HTML document.

    • For Example

    Code :
    <style>
         body {
            background-color:lightblue;
        }
    </style>


  • Embedded style sheets are useful when you want to apply styles to a specific HTML document without affecting other documents on your website.
