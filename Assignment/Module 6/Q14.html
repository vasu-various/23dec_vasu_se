Q.14 What is the use of a span tag? Explain with example?

  • span tag is inline element
  • The tag is used for a give a style in inlin text

    • For xampl :

    This is a < span style="color: blue;">blue< /span> apple.

    Output : This is a blue apple.

  • here we put blue word in span tag so we can give a style in spcific word using inline element.
