Q.8 What are the various formatting tags in HTML?

  • HTML provides a variety of formatting tags that can be used to apply formatting to the content of a web page.

    • < b> and < strong>: Used to make text bold.
    < i> and < em>: Used to italicize text.
    < u>: Used to underline text.
    < s>: Used to strike through text.
    < sub>: Used to make text appear as subscript.
    < sup>: Used to make text appear as superscript.
    < small>: Used to make text appear smaller.
    < big>: Used to make text appear bigger.
    < code>: Used to display code or programming language syntax.
