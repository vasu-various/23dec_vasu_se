Q.11 What is the difference between a block-level element and an inline element?

  • block-level elements and inline elements are two types of elements in html
  • block level element contain a 100% width of a web page

    • Example of block level element is < p> < li> < h1>

  • inline element cantain a width as a content size

    • Example of inline element is < span>

  • every block level element start in new line
