Q.7 What is the difference between the ‘id’ attribute and the ‘class’ attribute of HTML elements?

  • Both the id and class attributes are used to specify the styling and behavior of elements.
  • id can not be same for more than one element
  • class can be same for more than one element

    • For example:

    .v{
    background-color: blue;
    }

    #v{
    background-color: blue;
    }

  • first . is for class selector and # is for a id selection
