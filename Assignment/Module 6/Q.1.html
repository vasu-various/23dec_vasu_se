Q. 1 - Are the HTML tags and elements the same thing?

  • No, HTML tags and elements are not the same thing, but they are related.
  • the element is a group of start tag, content and the end tag

    • for Example :

    <start-tag> Content <end-tag>

  • tag comnly has a start and end tag
  • self tag has only one tag

    • for Example :

    <hr>, <br>
