Q.20 What are logical and physical tags in HTML?

  • In HTML We have logical tags and physical tags.
  • Logical tags are used to provide meaning and structure to the content on a web page.

    • Examples :
    <header>, <footer>

  • Physical tags are used to presentation of the content on a web page.

    • For Example :
    <b>, <i>, <u>, <center>, <img>, <table> and <hr>.
