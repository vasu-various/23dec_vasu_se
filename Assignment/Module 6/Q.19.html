Q.19 What is difference between HTML and XHTML?

  • HTML stand for Hypertext Markup Language
  • XHTML stand for Extensible Hypertext Markup Language
  • Both are the markup languages that used to create web pages.
  • The main difference between HTML and XHTML is that HTML is forgiving errors but XHTML has a strict that requires proper structure.
  • XHTML is case sensitive but HTML is not a case sensitive
