Q.15 How to insert a picture into a background image of a web page?

  • We can insert a image in background using a css
  • In your CSS file, use the background-image property ith slecting body tag

    • body {
         background-image: url('background-image.jpg');
         background-size: cover;
    }

    Other Way :
    <body background="image/background.jpg">

  • here we saw a two ways of set a background image
