Q.11 What are all the looping structures in JavaScript?

  • In JavaScript, there are several looping structures you can use to repeatedly execute a block of code. Here are the main looping structures:
  • We have two type of looping structures 1. Entry control loop 2. Exit Control loop
  • Entry control loops
  • for loop:
    The for loop allows you to specify an initialization, a condition, and an increment/decrement, and it repeatedly executes a block of code while the condition is true.
  • while loop:
    The while loop repeatedly executes a block of code as long as a specified condition is true.
  • for...in loop:
    The for...in loop iterates over the enumerable properties of an object. It's primarily used for iterating over the keys of an object.
  • for...of loop:
    The for...of loop is used to iterate over the values of iterable objects (arrays, strings etc.)
  • forEach loop:
    The forEach method is available for arrays and is used to execute a provided function once for each array element.

  • Exit Control loop
  • do...while loop:
    The do...while loop is similar to the while loop, but it guarantees that the block of code is executed at least once before checking the condition.
