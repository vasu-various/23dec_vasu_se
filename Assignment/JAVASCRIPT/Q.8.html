Q.8 What is === operator?

  • In JavaScript, the === operator is used for strict equality comparison.
  • It compares two values to check if they are equal in both value and data type.
  • This means that not only the values need to be the same, but the types of the values must also match.
  • It's generally recommended to use the === operator (strict equality) instead of == to avoid unexpected type coercion and ensure more accurate comparisons.
