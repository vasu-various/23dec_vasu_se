Q.16 How can a page be forced to load another page in JavaScript?

  • In JavaScript, you can use the window.location object to redirect the current page to a different URL, effectively loading another page.
  • If you want to load a different page after a certain delay or in response to some event, you can use functions like setTimeout() or handle events accordingly.
  • Remember that using JavaScript to forcefully redirect a user can potentially disrupt their experience, so make sure to use this technique responsibly and consider user experience.
