Q.5 What are undeclared and undefined variables?

  • In JavaScript, "undeclared" and "undefined" are terms used to describe different states of variables.
  • Undeclared Variables:
  • An undeclared variable is a variable that has been used in code without being declared using the var, let, or const keyword.
  • When you try to use an undeclared variable, JavaScript will show a ReferenceError.

  • Undefined Variables:
  • An undefined variable is a variable that has been declared but has not been assigned a value.
  • If you encounter an "undefined" error, it often means that you've declared a variable but haven't assigned a value to it yet.
