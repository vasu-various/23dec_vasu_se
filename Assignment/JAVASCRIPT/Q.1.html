Q.1 What is JavaScript?

  • JavaScript is a widely used programming language primarily known for its ability to add interactivity, dynamic behavior, and functionality to websites.
  • It was originally created by Brendan Eich at Netscape Communications in 1995
  • JavaScript enables the modification of webpage content without needing to reload the entire page.
  • JavaScript can interact with the DOM, which represents the structure and content of a webpage.
  • JavaScript allows developers to define actions that should be taken when certain events occur, such as a button click, form submission, or mouse movement.
