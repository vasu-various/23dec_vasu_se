Q.2 What is the use of isNaN function?

  • The isNaN function is a built-in JavaScript function used to determine whether a value is NaN (Not-a-Number)
  • The isNaN function takes a single argument and returns a Boolean value:
  • If the argument is NaN (i.e., not a valid number), the function returns true.
  • If the argument is a valid number, the function returns false.
  • In summary, the isNaN function in JavaScript is used to check if a value is NaN or not
