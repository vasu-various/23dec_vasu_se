Q.4 Canvas and SVG tags

  • SVG :
  • SVG is a stand for Scalable vector Graphics
  • SVG defines vector-based graphics for the Web
  • SVG defines the graphics in XML format
  • SVG images can be printed with high quality at any resolution
  • SVG graphics do NOT lose any quality if they are zoomed or resized

  • Canvas :
  • Canvas is used to draw graphics on a web page.
  • Canvas defines Raster based graphics for the Web
